Color Picker

I'm sorry; but it appears that you don't have WebGL installed or emabled in your browser. Please download an updated browser; or enable it in your settings.
This colour picker is rendered entirely in the browser using WebGL. I think it looks pretty cool.
The problem is that the highlighted areas look terrible especially when viewed from anything other than straght on to the screen.
The solution was to modify the rendering of the color selector to create a flatter transition into the secondary colors.
This creates a much nicer looking color picker.
The code to make these spinners using WebGL was made with the help of the Introduction to WebGL tutorial(it's really good). The color picker uses the HSV algorithm from Wikipedia and was mostly developed during CampJS.
